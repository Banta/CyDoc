Erste Schritte

Bei CyDoc anmelden

Öffnen Sie die folgende Internet-Adresse mit dem Firefox (empfohlen) oder einem anderen Webbrowser https://cydoc-demo.zytolabor.ch

Geben Sie auf der Anmeldeseite Ihren Benutzernamen und Ihr Passwort ein.

Bestätigen Sie dann mit „Anmelden“

Um unsere Software ein bisschen kennen zu lernen, können Sie mit den folgenden Login-Daten auf unsere Test-Datenbank zugreifen. Dort darf mit allen Daten gespielt werden

Login: test

Password: Example

Patient suchen

Rechts oben befindet sich stets ein Suchfeld, um Patienten zu suchen. Geben Sie im Suchfeld einen Teil des Namens (es funktioniert auch mit dem Vornamen) oder das Geburtsdatum der gewünschten PatientIn ein und drücken Sie dann „Enter“ um die PatientIn zu suchen.







Danach erscheint im unteren Bereich die Liste aller gefundenen PatientInnen, welche den Suchbegriff enthalten. (Falls Sie nach „ülle“ suchen, wird auch Frau Meier-Müller gefunden). Um die Details anzuzeigen klicken Sie einfach auf die gewünschte PatientIn.





Die Personalien werden auf der Hauptseite angezeigt. Um die Personalien zu ändern, klicken Sie bitte auf „Ändern“. Die Daten werden dann editierbar. (Auf der nächsten Seite näher beschrieben)

Für kleine Änderungen an Personendaten (z.B. Namen oder Adresse falsch geschrieben), kann das fehlerhafte Feld direkt angeklickt werden.

Jetzt kann man das Feld editieren und mit „OK“ speichern.

Mit „Cancel“ kann man das Editieren abbrechen.

Patient bearbeiten

Öffnen Sie eine PatientIn und klicken Sie dann auf „Ändern“. Danach erscheint die Eingabemaske zum korrigieren der Personalien. Klicken Sie nach der Korrektur auf „speichern“ um wieder auf die Patienten-Ansicht zu gelangen.

Neue Patienten anlegen

Klicken Sie im Hauptmenü auf Patienten und im nun geöffneten Fenster auf „Erfassen“. Eine Maske zur Eingabe der Personalien öffnet sich. Schliessen Sie dann mit „Speichern“ ab.




Rechnungswesen

Fall eröffnen

Öffnen Sie eine PatientIn und klicken Sie anschliessend oben rechts auf „Fall eröffnen“. Geben Sie dann die nötigen Informationen wie Anfangsdatum und Behandlungsgrund an und beenden mit „speichern“

Leistung erfassen

Öffnen Sie einen Patienten und klicken Sie links auf Fälle. Der aktuelle Fall wird angezeigt und Sie können mit dem Erfassen von Leistungen (Tarmed, Analysenliste, Medikamente) beginnen. Klicken Sie erst auf den Link „Neue Konsultation“. Nun können Leistungen anhand der Tarifnummer oder der Beschreibung gesucht werden. Wenn nur eine Leistung zum Suchbegriff gefunden wird, übernimmt CyDoc diese automatisch. Falls mehrere Leistungen gefunden werden, müssen Sie eine mit der Maus anklicken.

Leistung löschen

Um eine Leistung zu löschen, klicken Sie einfach Rechts neben dem Text auf das rote Kreuz.

Diagnose erfassen

Im Suchfeld unter „Diagnosen“ können Sie nach Diagnosen suchen. Wenn die Suche eindeutig ist, wird die Diagnose automatisch erfasst, falls es mehrere Treffer gibt, müssen Sie eine Position wählen. Selbstverständlich können Sie nach den Diagnosecodes oder auch nach dem Diagnosetexten suchen. Um weitere Diagnosen eingeben zu können klicken Sie auf „neue Diagnose“.

Rechnung erstellen

Um aus den erfassten Leistungen eine Rechnung zu generieren, klicken Sie auf „Rechnung stellen“.

Ein neues Formular wird geöffnet, bei dem Sie das Rechnungsdatum, den Garanten und nach Wunsch auch noch Bemerkungen angeben können. Erstellen Sie nun die Rechnung indem Sie auf „Erstellen“ klicken.











Sie werden dann nach und nach über den Status informiert und gefragt, ob die Rechnung gedruckt werden soll und ob die Rechnung verbucht werden soll. Antworten Sie entsprechend jeweils mit „Ja“ oder „Nein“.

Rechnungskopie drucken

1.Wählen Sie einen Patienten aus (wenden Sie dazu die Suche an, welche weiter oben genauer beschrieben ist.)

2.Im linken Menu auf den Link „Rechnungen“ klicken.

3.Im Fenster erscheint die aktuellste Rechnung. Um eine ältere Rechnung auszuwählen, klicken Sie rechts auf die gewünschte Rechnung.

4.Um die Rechnungskopie zu drucken, klicken Sie auf „Rechnungskopie drucken“.

Zahlung verbuchen

mit VESR/BESR

Laden Sie die VESR-Datei von Ihrer Bank herunter und merken Sie sich, in welchen Ordner Sie die Datei gespeichert haben.




Gehen Sie dann in CyDoc (Home-Site) auf „VESR-Liste“und klicken Sie auf „VESR Datei einlesen“. Danach klicken Sie bitte auf Durchsuchen, wählen die richtige Datei und bestätigen mit „Öffnen“. Wenn Sie nun auf Einlesen klicken, wird die Datei hoch geladen und verbucht. Als Bestätigung wird ein Protokol angezeigt, welches Sie bei bedarf ausdrucken können

manuell verbuchen

Um manuell eine Zahlung zu verbuchen, öffnen Sie bitte die betreffende Rechnung. Klicken Sie dann unter „Buchungen“ auf „Manuell buchen“. Geben Sie dann das Zahlungsdatum, die Zahlungsart, optional eine Bemerkung und den Betrag an. Dann klicken Sie abschliessend auf „Buchen“.

Mahnwesen

Fällige Mahnungen anzeigen

Um die fälligen Mahnungen anzuzeigen, klicken Sie oben auf „Rechnungen“. Unter dem Titel „Überfällige Rechnungen“ sind alle Rechnungen aufgelistet, welche gemahnt werden sollen, oder bereits gemahnt wurden.

Mahnungen drucken

Öffnen Sie zum Drucken einer Mahnung die betreffende Rechnung und klicken Sie auf „Mahnung drucken“. Die Mahnung wird dann erstellt und direkt ausgedruckt.

Wenn man alle Mahnungen auf einmal ausdrucken möchte, geht man auf „Rechnungen“. Rechts oben hat man nun einen Link „Alle Mahnungen drucken“. Diesen wählt man aus.

Stammdaten

Versicherung suchen

Um eine Versicherung zu suchen klicken Sie einfach im Menü auf „Versicherungen“ und geben dann den Namen oder einen Teil davon im Suchfeld ein und bestätigen mit „Enter“.

Tarifposition suchen

Um Tarifpositionen zu suchen können Sie im Menü auf „Leistungen“ klicken und dann nach Tarif-Codes oder Beschreibungstext suchen.

(Diese Angaben entsprechen genau den Ergebnissen, welche Sie bei der Suche in „Leistungen erfassen“ erhalten.)

Medikamente suchen

Um Medikamente zu suchen, klicken Sie im Menü auf „Medikamente“. Sie können dann nach dem Namen des Präparates suchen. Um die Details eines Medikamentes einzublenden, klicken Sie es einfach mit der Maus an.

Ärzte suchen und bearbeiten

Um Ärzte zu suchen, klicken Sie im Menü auf „Ärzte“. Sie können dann nach dem Namen des Arztes suchen.

Mit „Erfassen“ können Sie einen neuen Arzt im System eingeben.

Wenn Sie die Daten ändern möchten, klicken Sie auf den gewünschten Arzt. Nun können Sie wieder rechts oben „Ändern“ auswählen oder direkt durch Klick auf das entsprechende Felder überarbeiten.

Tips und Tricks

Mehrere Fenster

CyDoc ist so programmiert, dass man beliebig viele Fenster geöffnet haben kann, zum Beispiel eines für die Rechnungen, eines für die Patientenverwaltung und eines für die Medikamenten-Suche. Um einen Menü-Punkt in einem neuen Fenster zu öffnen, klicken Sie mit der rechten Maustaste auf den gewünschten Menü-Punkt und klicken dann auf „Link in neuem Fenster öffnen“.

Lesezeichen

Da CyDoc im „Internet“ läuft, kann man leicht Lesezeichen (Favoriten) anlegen. Zum Beispiel für einen Fall, den man sich für die spätere Bearbeitung merken möchte. Klicken Sie hierfür mit der rechten Maustaste auf den gegebenen Fall und wählen Sie „Lesezeichen für diesen Link hinzufügen“.

Es öffnet sich dann ein kleines Fenster, in dem Sie angeben können, ob das Lesezeichen auf der Symbolleiste oder im Lesezeichen-Menü gespeichert werden soll. Des weiteren können Sie einen Namen für dieses Lesezeichen angeben und mit „Fertig“ abschliessen.
